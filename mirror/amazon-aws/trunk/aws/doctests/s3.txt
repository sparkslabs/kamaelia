########################################
Module :mod:`aws.s3`
########################################

The :mod:`~aws.s3` module had components for getting :class:`boto.s3.key.Key` objects from an :class:`boto.s3.bucket.Bucket` and vice versa. All of the base classes are located in :mod:`aws.s3.base` and all of the actual components are in :mod:`aws.s3.components`. All of the components are also available in the :mod:`aws.s3` namespace

--------------------------------------

This module contains all of the superclasses for the components in the :mod:`~aws.s3.components` module
