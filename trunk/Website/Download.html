Downloads

Kamaelia is divided into two portions - Axon and Kamaelia. Axon forms the core concurrency toolkit, whereas Kamaelia is a collection a components that use Axon to run. You need BOTH in order to run Kamaelia based systems. (Think of Axon like a kernel and Kamaelia as user programs)

  • Kamaelia's main release currently stands at 0.1.2, and requires Axon 1.1.0 or above for full functionality.
  • Axon's main release currently stands at 1.1.0

Europython Distribution

For Europython we produced a Kamaelia 0.2.0 pre-release version to accompany the presentation and slides. This distribution is still on the Europython site here: Kamaelia_EP2005.tar.gz. Inside you will find the following:

  • A simple presentation tool written in Kamaelia
  • The bulk of the presentation as slides in a Slides directory
  • The tar ball of Axon-1.1.1-ep.tar.gz - which is a pre-release version of 1.1.1
  • The tar ball of Kamaelia-0.2.0-ep.tar.gz - which is a pre-release version of Axon 0.2
