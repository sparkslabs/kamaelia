Kamaelia.Physics.Behaviours.loopingCounter

class loopingCounter(Axon.Component.component)

loopingCounter(increment[,modulo]) -> new loopingCounter component

Emits an always incrementing value, that wraps back to zero when it reaches the specified limit.

Keyword arguments: - increment -- increment step size - modulo -- counter wrap back to zero before reaching this value (default=360)

Inboxes

  • control : Secondary inbox often used for signals. The closest analogy is unix signals
  • inbox : Default inbox for bulk data. Used in a pipeline much like stdin

Outboxes

  • outbox : Default data out outbox, used in a pipeline much like stdout
  • signal : The default signal based outbox - kinda like stderr, but more for sending singal type signals
