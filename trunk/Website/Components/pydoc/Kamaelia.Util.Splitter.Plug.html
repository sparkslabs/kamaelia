Kamaelia.Util.Splitter.Plug

class Plug(Axon.Component.component)

A 'plug' for getting another component to 'plug-into' a splitter.

A splitter is any component with a 'configuration' inbox that will accept addSink and removeSink messages.

This component provides encapsulation for that which you wish to plug in.

Usage:

mysplitter = PlugSplitter(...) plug = Plug(mysplitter, Destination)

mysplitter.activate() plug.activate()
