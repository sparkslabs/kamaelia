Kamaelia.Util.Marshalling.DeMarshaller

class DeMarshaller(Axon.Component.component)

DeMarshalls (serialises) data presented on the inbox.
Use this component by providing an object with a fromString method that will convert (deserialise) the data from a string to its original form
