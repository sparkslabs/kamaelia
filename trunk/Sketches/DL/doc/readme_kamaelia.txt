Introduction
============

Kamaelia is a library of components that helps in building concurrent systems. These components are built on top of the Axon Framework.

Documentation
=============

The documentation of Kamaelia is available in the "Docs" subdirectory.
Additional Documentation can be found online at http://kamaelia.sourceforge.net/Docs/

To understand the basics of the Kamaelia system, a tutorial is available at http://kamaelia.sourceforge.net/MiniAxon/

Examples
========

The "Examples" subdirectory contains simple systems implemented using Kamaelia. 
Please check out the README in the Examples directory for further description of each example.
Additional cookbook systems are available online at http://kamaelia.sourceforge.net/Cookbook.html
