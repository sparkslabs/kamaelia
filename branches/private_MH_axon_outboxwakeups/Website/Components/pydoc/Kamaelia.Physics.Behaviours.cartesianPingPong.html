Kamaelia.Physics.Behaviours.cartesianPingPong

class cartesianPingPong(Axon.Component.component)

cartesianPingPong(point,width,height,border) -> new cartesianPingPong component

A component that emits (x,y) values that bounce around within the specified bounds.

Keyword arguments: - point -- starting (x,y) coordinates - width, height -- bounds of the area - border -- distance in from bounds at which bouncing happens

Inboxes

  • control : Secondary inbox often used for signals. The closest analogy is unix signals
  • inbox : Default inbox for bulk data. Used in a pipeline much like stdin

Outboxes

  • outbox : Default data out outbox, used in a pipeline much like stdout
  • signal : The default signal based outbox - kinda like stderr, but more for sending singal type signals
