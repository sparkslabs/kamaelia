Kamaelia.Protocol.FortuneCookieProtocol.FortuneCookieProtocol

class FortuneCookieProtocol(Axon.Component.component)

FortuneCookieProtocol([debug]) -> new FortuneCookieProtocol component.

A protocol that spits out a random fortune cookie, then terminates.

Keyword arguments: - debug -- Debugging output control (default=0)

Inboxes

  • control : Secondary inbox often used for signals. The closest analogy is unix signals
  • inbox : Default inbox for bulk data. Used in a pipeline much like stdin

Outboxes

  • outbox : Default data out outbox, used in a pipeline much like stdout
  • signal : The default signal based outbox - kinda like stderr, but more for sending singal type signals
